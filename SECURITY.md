# Security Policy

## Supported versions

Cambium is pre-1.0 and under active development. Only the latest `main` is
supported; there are no released versions yet.

## Reporting a vulnerability

Please report security issues **privately** via GitHub's
[private vulnerability reporting][report] — the **Report a vulnerability** button
on the repository's **Security** tab — rather than opening a public issue.

You can expect an acknowledgement of your report, and we will coordinate a fix
and disclosure timeline with you.

[report]: https://github.com/Nagato-Yuzuru/cambium/security/advisories/new
